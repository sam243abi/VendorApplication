File Structure

When you unzipped the files after downloading , You will find HTML folder that have the code's files.

|── HTML/
│   ├── assets/
│   │   ├── css/
│   │   ├── images/
│   │   ├── js/
│   │   ├── scss/
|   |   |   ├── base/
|   |   |   ├── breakpoints/
|   |   |   ├── demos/
|   |   |   ├── global/
|   |   |   ├── layout/
|   |   |   ├── modules/
|   |   |   ├── sections/
|   |   |   ├── vendor/  
│   ├── php/
│   │   ├── contact.php

assets : that has the assets files of the theme css, js,scss and images

php : that has the contact php file

Scss CSS-preprocessor know more

  1. Base : BasicRules , Helpers classes and Basic Pages Animations
  2. Breakpoints : Contains responsive and breakpoints styling
  3. Demos : Contains the styles of the demos ( Add the demo class to body tag)
  4. Global : Variable file along with starting point Mixins and Placeholders
  5. Layout : Contains styling for larger layout components e.g. nav, header, footer,etc
  6. Modules : Re-usable site elements e.g. buttons ,lists ,tabs ,etc
  7. Sections : Contains page-specific and sections styling
  8. Vendor : Default fall-backs and external files.
